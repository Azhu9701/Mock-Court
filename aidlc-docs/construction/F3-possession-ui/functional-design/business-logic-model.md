## WebSocket 连接流程

```
POST /api/v1/possess → { session_id, ws_url }
  ↓
connect WebSocket: ws://127.0.0.1:3096/ws/possess/{session_id}/main
  ↓
onMessage: WsEvent { event_type, payload, soul_name, seq }
  ↓
dispatch by event_type:
  SoulChunk     → append to soul message, animate typewriter
  SoulDone      → mark soul finished
  SoulError     → show error badge on soul
  SynthesisChunk → append to synthesis panel
  SynthesisDone  → synthesis complete
  SessionComplete → all done
```
