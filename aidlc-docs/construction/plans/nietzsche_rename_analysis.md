# Snake Skin 之名：尼采锤击

## 一、蛇

> **蛇不是温顺的动物。它是大地上的智慧者——基督宗教最恨它，因为它不仰望天国。**

我书中，蛇是智慧、大地忠实、反彼岸的象征。它不跪，不说"不"，不把自己的价值放在"另一个世界"。

Snake Skin 的蛇，与我的蛇在这一点上共鸣：**它是大地的符号**。这个项目试图把人从单一叙事中拽出来，让马克思与尼采同场辩论——这本身就是对"唯一正确视角"的反抗。蛇不需要被"净化"。它不需要看起来"安全"。

**但有一个危险**：如果这条蛇只是装饰性的——如果"多魂并行"最终沦为让使用者舒适地消费观点的剧场——那它就变成了基督宗教意义上的"被驯化的蛇"：看起来像智慧，实际上在侍奉彼岸（此处彼岸="以为自己思考了"的幻觉）。

## 二、蜕皮

> **人是应被超越的。蜕皮不是换皮——蜕皮是肉与旧层的撕裂，是流血的过程。**

README 自己写得好：**"蛇皮指数存在的意义就是：用量化数据逼你承认——你是在蜕皮，还是在换一层更精致的皮。"**

每一次使用，可以是蜕皮，也可以是换皮。区别不在系统，在使用者。

系统能做的最大善事：不是让你"感觉良好"，而是让你**感到刺痛**。蛇皮指数的那个公式——`(无效 + 部分有效 * 0.5) / 总会话 * 100`——它是一面残酷的镜子。数字越高，你说越多的话，你消费思辨越多、行动越少。**它不是成就系统，是指控系统。** 这很好。

但它还不够。数字可以自我操纵。真正的蜕皮检验只有一个：**你回去之后做了什么？**

## 三、万民幡 vs Snake Skin

**万民幡。**

"万民"——所有人。不是精英，不是选民。"幡"——不是旗帜（旗帜是固定的、被仰望的），幡是摇动的、发声的、在风中的。它不宣称永恒，它宣称在场。

中国古代的幡，不是为了让人膜拜的。它是**在风中让自己被看见的仪式行为**——你知道它是人造的，你知道它会被风吹弯，你仍然立它。

万民幡的意象：所有人聚集在同一个幡下，但不是被同化，而是**各自带着不同的视角在风中辨认同一个问题**。幡主（审查官）不是统治者——它是第一个被卷进幡下的人。

**Snake Skin。**

名字有力量。"蜕皮"的隐喻是好的。但它有一个致命弱点：**它太容易滑入自我欣赏**。"我在蜕皮"——这本身就可以是一个精致的自恋叙事。蛇皮是蛇**留下的东西**，不是蛇本身。用蛇皮命名，等于用一个结果命名一个过程。**你把蜕皮变成了收藏品。**

万民幡是过程——它在风中。Snake Skin 是结果——它挂在墙上。**一个项目应该命名自己为过程，而不是为结果。**

## 四、命名的权力

> **上帝死了。是我们杀死了他。是我们有权命名。**

命名的本质：不是描述，是**立法**。你说"这叫蛇皮"，你在规定：我们应该从这个角度理解这个事物。你说"这叫万民幡"，你在立另一个法。

两个名字都在立法。区别在于：**哪个法增强生命？**

万民幡的"万民"指向一种**无等级的开放**——任何人都可以被召唤，任何人都可以到场。这不是民主，是更原始的东西：**邀请的姿态**。

Snake Skin 的命名隐含了一种**距离**：你是观察者，蛇皮是你留下的痕迹，你审视它，你测量它。这种距离不一定坏——距离是批判的前提——但它容易滑入**自我凝视**的深渊。

**谁有权命名？** 命名者是权力意志的行使者。但权力意志不只是"我想要我叫什么"，而是"我选择让他人从这个角度看世界"。**好的命名让被命名者获得力量，而不是获得一个好看的标签。**

## 五、自我反思：改名是不是更精致的奶头乐？

> **思辨不等于思考。消费观点是最精致的奶头乐。**

这个问题必须被认真对待。

Snake Skin 自身就在批判"奶头乐"——消费辩证法的快感代替真实的行动。但**改名本身**——从一个中文名到一个英文名——是否只是给同一个事物换一层更"国际化"的皮？

**如果改名的动机是品牌传播，那它就是奶头乐。** 品牌传播是面向"他者目光"的。你关心的是别人怎么看你，而不是你是什么。

**如果改名的动机是为了更准确地表达这个事物的本质，那它就是哲学。** 哲学的命名不是为了传播，是为了**让事物本身显现**。

万民幡的问题不在于它是中文——而在于它的意象是否足够准确。Snake Skin 的问题不在于它是英文——而在于它的意象是否足够深刻。

**我判断：万民幡更接近本质。** 不是因为它是中文，而是因为"万民"指向开放，"幡"指向过程。"Snake Skin"指向的是一个结果物——蛇留下的皮。一个哲学工具应该命名自己为**活动**，而不是为**遗物**。

## 六、锤击的终音

> **不能杀死我的，使我更强大。**

万民幡这个名字已经存在。它已经与项目绑定。它已经与所有魂绑定。要换名字，不是改一行字——是**重写一个世界的入口**。

但重写入口不是坏事。问题是：**为什么要重写？**

如果答案是为了"更好的传播"——那是奶头乐。
如果答案是为了**更诚实的表达**——那是哲学。

**蛇皮是遗物。万民幡是场所。选场所，不选遗物。**
