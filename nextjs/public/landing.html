Core Features

From semantic collision detection to dialectical synthesis — every feature serves the goal of deeper understanding.

⚡

Real-time Collision Detection

Three-path parallel: keyword rules + trigram semantic similarity + ismism coordinate distance. Detects contradictions, blind spots, and redundancies.

🎭

6 Possession Modes

Single soul, conference, debate, relay, learning, and practice modes. Each mode activates different interaction topologies and intervention patterns.

🕸️

Soul Memory Graph

Directed graph with premise→conclusion chains. BFS contradiction detection at O(n), zero LLM cost. Supports graph merging and blind spot discovery.

🐍

Snake Skin Index

The higher the number, the more you consume AI debates instead of acting. Using AI dialectics instead of your own thinking is the most refined form of tittytainment.

📡

@mention Soul Summoning

Type @soul_name in follow-up to summon any soul as a sub-agent. They respond with full identity and context awareness, not through synthesis.

🔍

SearXNG Integration

Self-hosted meta-search engine. Inject real-time web search results into soul context. Independent search page for general queries.

⚖️

Three-Tier Gate Control

L1 keyword rules (μs) → L2 trigram Jaccard (ms) → L3 Flash LLM (s). Intelligent intervention injection only when needed.

🏛️

Interrogation Injection

Before discussion starts, an interrogator questions the user. Their answers inject into the shared temporal context all souls reference.

🔄

Adaptive Topology

5 topologies: Minimal, ClusteredParallel, FullMesh, Oppositional, SequentialLadder. Auto-selects based on diversity + complexity + budget.
