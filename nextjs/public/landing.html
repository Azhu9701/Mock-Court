Features

Every feature serves one goal.

Deeper understanding through structured collision. Not more information — more perspectives.

⚡

Real-time Collision Detection

Three-path parallel detection: keyword rules (μs), trigram Jaccard (ms), ismism coordinate distance. Finds contradictions, blind spots, and redundancies as souls stream.

🎭

6 Possession Modes

Single, conference, debate, relay, learning, practice. Each activates different topologies and intervention patterns for different cognitive tasks.

🕸

Soul Memory Graph

Directed premise→conclusion chains per soul. BFS contradiction detection at O(n), zero LLM cost. Cross-soul graph merging and blind spot discovery.

🐍

Snake Skin Index

The higher the number, the more you consume AI debates instead of acting. Using dialectics instead of thinking is the most refined tittytainment.

@

@mention Soul Summoning

Type @soul_name to summon any soul as a sub-agent. They respond with full identity, not through synthesis. They know why they were summoned.

⚖

Three-Tier Gate Control

L1 keyword rules → L2 trigram Jaccard → L3 Flash LLM. Real-time intervention injection only when a genuine collision is detected.

Interrogation Injection

Before discussion, an interrogator questions you. Answers inject into shared temporal context all souls reference.

SearXNG Integration

Self-hosted meta-search. Inject real-time web results into soul context during follow-ups.

Adaptive Topology

5 topologies auto-selected by diversity, complexity, and budget. 30s collision-free auto-downgrade.
